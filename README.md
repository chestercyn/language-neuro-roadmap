# Language Neuroscience Roadmap

A personal learning-roadmap dashboard for building expertise in the neuroscience of language — how language arises in the human brain, how it is impaired by disease, and how to restore it.

Five concentric pillars across a 36-month horizon, with progress tracking, a Gantt timeline, an expert directory, and a learning log.

**Live dashboard:** https://chestercyn.github.io/language-neuro-roadmap/

Progress is stored in your browser's `localStorage`. Use the Export / Import JSON buttons to back up or sync between devices.
